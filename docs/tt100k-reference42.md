## 数字复核

远距离视频中的 `50` 容易被检测器误判为 `pl40`、`pl70`、`pl80`、`pl100`，有时还会误判为 `pr40`。视频工作流只对这些已复现且经过样本验证的候选调用项目已有的 GTSRB 裁剪分类器；仅当分类器判定为 50 且达到复核门槛时才替换为 `pl50`。普通候选使用 `VIDEO_SPEED_REFINEMENT_MIN_CONFIDENCE=0.90`；`pl40` 使用经过真实样本校准的 `VIDEO_SPEED_REFINEMENT_PL40_MIN_CONFIDENCE=0.75`。

该规则不会泛化到所有数字类别。对 TT100K 原始样本的交叉验证显示，GTSRB 对 `pl100`、`pl120`、限高和限质量标志存在明显域偏移；因此分类器给出 20、30、60、70、80、100 或 120 时不会覆盖 TT100K 检测结果，`pl5`、`pm*`、`ph*` 也不参与直接替换。100 个真实 `pl40` 样本中，仅 2 个被 GTSRB 判为 50，置信度最高为 `61.76%`，低于 `pl40` 专用的 `75%` 门槛；示例视频第 1329 帧的真实 50 则达到 `75.53%`。
