# Training Workspace

This folder stores local training data and model outputs.

- `raw/tt100k/data.zip`: official TT100K download.
- `raw/tt100k/data`: extracted TT100K images and annotations.
- `datasets/tt100k`: YOLO-format dataset generated by `backend/tools/prepare_tt100k.py`.
- `runs`: Ultralytics training outputs.

Useful commands:

```powershell
python backend\tools\prepare_tt100k.py `
  --annotations training\raw\tt100k\data\annotations.json `
  --image-root training\raw\tt100k\data `
  --output training\datasets\tt100k `
  --min-instances 100 `
  --image-mode hardlink

powershell -ExecutionPolicy Bypass -File scripts\train-tt100k-gpu.ps1
```
